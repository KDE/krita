Ten Scripts

A script to assign Python scripts to any of ten configurable hotkeys.

Basic Usage

Go to Tools → Scripts → Ten Scripts.

A Window will pop up with a list of boxes prepended by hotkey-names.

Press the last button to select a Python script from your system. It will then show up in the center label when set.

To change the associated hotkey, go to Settings → Configure Krita → Keyboard Shortcuts and then change the Execute Script actions under Ten Scripts

When done, open a file, and press your hotkeys to activate that script!
